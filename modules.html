data_preprocessing.load_and_clean_skills_6_2_data()[source]¶

Loads and cleans the Skills 6.2 data from an Excel file.

This function reads the “Table 6.2” sheet from the Excel file located in the ‘project_data/2023-33’ directory. It performs the following operations: 1. Skips the first row and uses the second row as the header. 2. Drops the last 6 rows of the dataset. 3. Removes the first row of the dataset and resets the index. 4. Splits the ‘2023 National Employment Matrix code’ column by the hyphen (‘-’) and keeps only the first part of the code.

Returns:

The cleaned DataFrame containing the data from the “Table 6.2” sheet with the appropriate transformations applied.

Return type:

pandas.DataFrame

Example

>>> df = load_and_clean_skills_6_2_data()
>>> df.head()
data_preprocessing.load_and_clean_skills_occupation_data()[source]¶

Loads and cleans occupation and skills data for 2023 and 2029.

This function reads and cleans occupation data for the years 2023 and 2029, along with skills data for 2023. It merges the occupation data on a common occupation code and calculates changes in employment between 2023 and 2029. The skills data is loaded from a specific sheet in an Excel file, and the data frames are cleaned and prepared for analysis. The function assumes that the required Excel files are present in the ‘project_data’ folder within the current working directory.

Returns:

  • common_occupation_df (pd.DataFrame): Merged and cleaned occupation data for comparison, with columns such as employment figures and occupational openings.

  • skills_6_1 (pd.DataFrame): Merged and cleaned skills data from the ‘Table 6.1’ sheet.

Return type:

tuple

Raises:

  • FileNotFoundError – If the required Excel files are not found in the expected paths.

  • ValueError – If there are issues with the structure or content of the data during cleaning.

Example

common_occupation_df, skills_6_1 = load_and_clean_skills_occupation_data() print(common_occupation_df.head()) print(skills_6_1.head())

Load and clean wage data from an Excel file.

This function reads an Excel file containing wage data, cleans the data by removing unnecessary columns, replacing special characters in specific columns, and converting the columns to numeric types. It also drops duplicates based on the ‘OCC_TITLE’ column.

Returns:

A cleaned DataFrame with selected columns from the wage data.

Return type:

pandas.DataFrame

Raises:

  • ValueError – If there are issues with data conversion during cleaning.

Example

data_preprocessing.process_wage_and_employment_data()[source]¶

Processes national wage and employment data from 2019 and 2023, merges the data, and returns a DataFrame with calculated employment changes and department information.

This function performs the following operations: 1. Reads the 2019 and 2023 national wage and employment data from Excel files. 2. Merges the data for both years based on occupation code and title. 3. Renames the columns for clarity. 4. Calculates the total employment change and the percentage change. 5. Extracts department information and adds the department number and name to the dataset. 6. Cleans the data by removing rows with certain patterns.

Returns:

A DataFrame containing merged and cleaned wage, employment data,

with employment changes and department information.

Return type:

pd.DataFrame

holland_code_model¶

holland_code_model.train_random_forest_model(df1)[source]¶

Trains a Random Forest classifier to predict the occupation code based on selected features, with feature engineering based on RIASEC themes, model evaluation, and top-5 accuracy.

This function performs the following steps: 1. Feature engineering: Adds RIASEC theme columns based on the mean of related columns. 2. Handles missing values by filling them with the column mean. 3. Creates a heatmap of feature correlation. 4. Splits the dataset using an 80-20 train-test split for each category. 5. Trains a Random Forest classifier. 6. Evaluates the model’s accuracy and top-5 accuracy. 7. Outputs the feature importance and top-5 predictions with their confidence levels.

Parameters:

df1 (pd.DataFrame) – DataFrame containing the employment data with various columns including: - Features related to RIASEC themes (e.g., ‘Mechanical’, ‘Physical strength and stamina’) - Target column: ‘2023 National Employment Matrix code’

Returns:

The function prints evaluation metrics including accuracy, feature importance, and top-5 accuracy.

Return type:

None

visualisation.create_interactive_job_prefix_plot(skills_6_1, top_5_codes)[source]¶

Creates an interactive plot for selecting job prefixes and displaying corresponding wage data.

This function generates a MultiSelect widget based on the top_5_codes list, which contains the prefixes of the 2023 National Employment Matrix codes. It then filters the skills_6_1 DataFrame based on these prefixes to show the corresponding job titles. The selected job prefixes used for a plot displaying the top 10 jobs by median wage.

  • codes. (job titles and corresponding matrix) –

  • top_5_codes (list of str) – A list of the top 5 matrix code prefixes used to filter

  • titles. (job) –

Returns:

A Panel layout containing the interactive widgets and plot.

Return type:

pn.Column

Example

skills_6_1 = pd.read_excel(“path_to_skills_data.xlsx”) top_5_codes = [‘15’, ‘17’, ‘11’, ‘13’, ‘23’] plot = create_interactive_job_prefix_plot(skills_6_1, top_5_codes) plot.servable()
